Python development
==================

* **Tab**: Auto complete and snippets

* **F6**: Run PEP8 for checking code

* **,v**: Display tasks list

* **F4**: set paste

* **F5**: set nopaste

* **>**: Indent code

* **<**: Deindent code

* **Gsdiff**: Execute *git diff*

* **Gcommit**: Execute *git commit*
